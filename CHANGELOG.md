# Changelog

All notable changes to "IDE Auto Retry" will be documented in this file.

## [1.0.0] - 2026-01-22

### Added
- Initial release
- Auto-click Retry buttons when AI agents encounter errors
- Chrome DevTools Protocol (CDP) integration
- Cross-platform support (macOS, Windows, Linux)
- Simple and clean UI
- Auto-start option
- Safety blocklist for dangerous commands
- Statistics tracking (retries count, connections)
- Activity log with timestamps
- Settings for customization (port, interval, cooldown)
